개인정보처리방침

시행일: 2025년 1월 1일 · 최종 수정: 2026년 4월 16일

CatchVoca(이하 "본 서비스")는 사용자의 개인정보를 소중히 여기며, 개인정보 보호를 위해 최선을 다하고 있습니다. 본 방침은 CatchVoca Chrome 확장 프로그램을 사용할 때 수집되는 정보와 그 처리 방식에 대해 설명합니다.

1. 수집하는 정보

CatchVoca는 서비스 제공을 위해 다음과 같은 정보를 수집합니다.

항목 수집 방법 저장 위치 목적
사용자가 저장한 단어 사용자 직접 저장 기기 내 IndexedDB 단어 학습 및 복습
복습 상태 (SM-2 알고리즘 데이터) 학습 진행 시 자동 생성 기기 내 IndexedDB 간격 반복 학습 스케줄 관리
앱 설정 정보 사용자 설정 변경 시 기기 내 Chrome 로컬 스토리지 사용자 환경 설정 유지
페이지 텍스트 (일시적) AI 분석 버튼 클릭 시 전송 후 즉시 폐기 단어 추천을 위한 AI 분석

로컬 우선 단어, 복습 상태, 설정 데이터는 기본적으로 사용자의 기기에만 저장됩니다. 서버로 전송되지 않으며, 오프라인 환경에서도 완전히 동작합니다.

2. 클라우드 동기화 (선택 사항)

클라우드 동기화는 선택적 기능이며, 활성화하려면 Google 계정으로 로그인이 필요합니다.

  • 동기화를 활성화하면 단어 및 학습 데이터가 Firebase(Google)에 암호화되어 저장됩니다.
  • 저장된 데이터는 사용자의 Google 계정에 연결되며, 동일 계정으로 로그인한 다른 기기에서 접근할 수 있습니다.
  • 로그아웃 시 클라우드 연결이 해제되며, 로컬 데이터는 유지됩니다.
  • Firebase 데이터 처리에 관한 자세한 사항은 Google Firebase 개인정보처리방침을 참조하세요.

3. AI 분석 기능 (Google Gemini)

중요 AI 페이지 분석 기능은 사용자가 직접 "분석 시작" 버튼을 클릭할 때만 동작합니다. 자동으로 페이지 내용을 수집하거나 전송하지 않습니다.

AI 분석 기능 사용 시 다음 사항을 유의하세요.

  • 분석 시작 버튼 클릭 시, 현재 페이지의 텍스트 내용이 Google Gemini API로 전송됩니다.
  • 전송된 텍스트는 단어 추천 분석 후 즉시 처리되며, CatchVoca 서버에 저장되지 않습니다.
  • Google의 AI 모델 정책에 따라 전송된 데이터가 처리될 수 있습니다. 자세한 사항은 Google 개인정보처리방침을 참조하세요.
  • 개인 정보가 포함된 페이지에서 AI 분석 기능 사용 시 주의하시기 바랍니다.

4. 모바일 퀴즈 기능

모바일 퀴즈 링크를 생성할 때 다음과 같이 데이터가 처리됩니다.

  • 복습 대상 단어 목록이 Firebase에 임시 업로드됩니다.
  • 생성된 링크는 일정 시간 후 자동으로 만료되며, 만료된 데이터는 삭제됩니다.
  • 모바일 퀴즈 완료 후 학습 결과가 확장 프로그램으로 동기화되고, Firebase의 임시 데이터는 삭제됩니다.

5. 제3자 정보 공유

원칙 CatchVoca는 사용자의 개인 데이터를 제3자에게 판매하거나 공유하지 않습니다.

단, 다음의 경우 외부 서비스와 데이터가 연동됩니다.

  • Google Firebase: 클라우드 동기화 및 모바일 퀴즈 기능 (사용자 동의 시)
  • Google Gemini API: AI 분석 기능 (사용자가 버튼 클릭 시)
  • 사전 API: 단어 뜻 조회 시 단어 텍스트만 전송 (Naver Dictionary, Free Dictionary API)

법적 요구, 사기 방지, 서비스 보호를 위해 필요한 경우 관련 법령에 따라 공개될 수 있습니다.

6. 쿠키 및 추적 기술

CatchVoca 확장 프로그램은 쿠키를 사용하지 않습니다.

사용자 활동을 추적하는 광고 네트워크 또는 분석 도구를 통한 추적을 수행하지 않습니다. 앱 내 데이터는 Chrome의 chrome.storage.local 및 브라우저 내 IndexedDB에만 저장됩니다.

7. 데이터 보관 및 삭제

  • 로컬 데이터(단어, 복습 상태, 설정)는 사용자가 확장 프로그램을 삭제할 때까지 기기에 보관됩니다.
  • 확장 프로그램 제거 시 모든 로컬 데이터가 삭제됩니다.
  • 클라우드 동기화 사용 시, 계정 삭제 또는 클라우드 데이터 삭제 요청으로 Firebase 저장 데이터를 삭제할 수 있습니다.
  • 데이터 삭제 요청은 아래 연락처로 문의하세요.

8. 아동 개인정보 보호

CatchVoca는 만 14세 미만 아동을 대상으로 하지 않습니다. 만 14세 미만 사용자는 보호자의 동의 하에 서비스를 이용해야 합니다. 아동의 개인정보가 무단으로 수집되었음을 발견한 경우, 즉시 삭제 조치를 취합니다.

9. 개인정보 보안

  • 모든 외부 API 통신은 HTTPS(TLS)를 통해 암호화됩니다.
  • Firebase에 저장되는 데이터는 Firebase Security Rules에 의해 사용자별로 접근이 제한됩니다.
  • 로컬 데이터는 Chrome의 보안 모델에 의해 다른 확장 프로그램 및 웹사이트로부터 격리됩니다.

10. 개인정보처리방침 변경

본 방침은 변경될 수 있습니다. 중요한 변경 사항이 있을 경우, 확장 프로그램 업데이트 시 안내드리거나 이 페이지에 공지합니다. 변경된 방침은 게시 즉시 효력이 발생합니다.

최신 방침은 이 페이지에서 항상 확인하실 수 있습니다.
